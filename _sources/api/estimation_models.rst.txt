Estimation Models
-----------------

The package ships with parametric estimation models covering the most common univariate distributions. Each model can be selected by string name in :class:`~regmmd.MMDEstimator` (e.g. ``model="gaussian-loc"``) or by passing a class instance directly. ``par_v`` denotes the variable parameter(s) that are optimised; ``par_c`` denotes constant parameter(s) that are held fixed. Click a model name for its full reference page.
